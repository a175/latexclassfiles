LaTeXのdocument classのリスト.

\documentclassで指定できるclsファイルのリスト (網羅的ではない).

用途による分類

articleやbookなど用途による分類.

長い文章を書くためのbookや, 短めの論文を書くためのarticleなどは有名ですが, 他にも色々な用途のクラスがあります. 詳しくはこちら.

パッケージによる分類

複数の文章構造のあるパッケージ/シリーズなど.

たとえば, AMSからは, amsart, amsbookなど複数のクラスが提供されています. 詳しくはこちら.
